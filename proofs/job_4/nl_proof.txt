Let n be a natural number. The first n odd natural numbers are

1, 3, 5, …, 2n−1.

Thus the conjecture asserts that

1 + 3 + 5 + ⋯ + (2n−1) = n².

We prove this by mathematical induction.

**Key idea**

The k-th odd natural number is 2k−1. Therefore the sum of the first n odd natural numbers is

∑_{k=1}^{n} (2k−1).

The goal is to show that this sum equals n².

**Proof by induction**

**Base case**

For n=1, the first 1 odd natural number is 1. Hence

1 = 1².

So the claim is true for n=1.

**Inductive hypothesis**

Assume that for some natural number n≥1, the sum of the first n odd natural numbers is n². That is, assume

1 + 3 + 5 + ⋯ + (2n−1) = n².

We must prove that the sum of the first n+1 odd natural numbers is (n+1)².

**Inductive step**

The first n+1 odd natural numbers are

1, 3, 5, …, 2n−1, 2(n+1)−1.

The last term is

2(n+1)−1 = 2n+1.

Therefore,

1 + 3 + 5 + ⋯ + (2n−1) + (2n+1)

is the sum of the first n+1 odd natural numbers.

By the inductive hypothesis,

1 + 3 + 5 + ⋯ + (2n−1) = n².

Substituting this into the previous expression gives

1 + 3 + 5 + ⋯ + (2n−1) + (2n+1) = n² + (2n+1).

But

n² + 2n + 1 = (n+1)².

Hence

1 + 3 + 5 + ⋯ + (2n−1) + (2n+1) = (n+1)².

So the statement is true for n+1.

**Conclusion**

The claim is true for n=1, and whenever it is true for n, it is also true for n+1. Therefore, by the principle of mathematical induction,

1 + 3 + 5 + ⋯ + (2n−1) = n²

for every natural number n≥1.

If one includes 0 as a natural number, the statement also holds for n=0, since the empty sum is 0, and

0 = 0².

## Final Verdict

VERDICT: PROVED

The conjecture is true for all natural numbers n: the sum of the first n odd natural numbers is exactly n².
